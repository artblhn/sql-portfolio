Про проєкт

User Behavior Dataset

Цей проєкт демонструє навички аналізу даних за допомогою SQL. Датасет містить інформацію про поведінку 700 користувачів мобільних пристроїв, включаючи час використання додатків, тривалість роботи екрану, використання даних та інші метрики.

Кожне завдання розкриває різні аспекти аналізу: від базової статистики до складних сегментацій користувачів. Ви можете редагувати SQL запити та бачити результати в реальному часі.

Про використання Google BigQuery

Всі SQL запити в цьому проєкті були розроблені та протестовані в Google BigQuery з використанням Standard SQL. Для демонстрації на цьому сайті запити були адаптовані під SQLite, що може призвести до незначних відмінностей у синтаксисі. Оригінальні запити для BigQuery доступні для завантаження.
